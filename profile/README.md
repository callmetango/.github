# Sonic Desktop Environment (SonicDE)

The community-driven Sonic Desktop Environment aims for keeping and improving the X11-specific ascpects of KDE. Currently it consists of the customized KWin/X11 and Plasma Desktop components.

## Contacting the SonicDE development team

* IRC: #sonicde on irc.libera.chat
* Telegram: https://t.me/sonic_de

## Support

### Application Developer

If you are an application developer having questions regarding windowing systems (X11) please do not hesitate to contact us. Preferably through our Telegram.

### End user

Please contact the support channels of your Linux distribution for user support. The Sonic DE development team does not provide end-user support.

## Reporting bugs

Please use the issue tracker in the appropriate repository to report your issue.
